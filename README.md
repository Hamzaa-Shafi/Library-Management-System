# 📚 Library Management System

A console-based **Library Management System** developed in **C++** using Object-Oriented Programming (OOP) and File Handling. The application helps manage library books, borrowing records, and fine calculations through a simple menu-driven interface.

## 🚀 Features

- ➕ Add new books
- 📖 Display all books
- 🔍 Search books by ID or title
- 📚 Issue books to borrowers
- ↩️ Return issued books
- ⏰ Automatic late fine calculation
- 📄 Report returned books as Torn or Lost
- 💰 Damage fine calculation
- 💾 Persistent data storage using text files
- 🖥️ Menu-driven console interface

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Dynamic Memory Allocation


## 📁 Project Structure

```
LibraryManagementSystem/
│── LibraryManagementSystem.cpp
│── books.txt
│── records.txt
│── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in Visual Studio or any C++ IDE.
3. Build the project.
4. Run the executable.
5. Ensure `books.txt` and `records.txt` are present in the project directory.
